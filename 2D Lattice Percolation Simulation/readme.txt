This simulates a simplified site percolation of a two-dimensional square lattice. In this simulation, a finite LxL square lattice was used, where L = 1024, and
the simulation was run on n = 100 randomizations, and the critical parameter p_c was estimated. The script generates the following plots: the intensity plots of the clusters
in the subcritical (p < p_c), critical (p \approx p_c), and supercritical (p > p_c) regions, the log-log plot of the probability density function of the non-largest clusters
as a function of cluster size around the critical parameter p_c fitted with a power law distribution, the plots of the mean counts of the largest, as well as the non-largest
cluster sizes as a function of occupation probability, and the plot of percolation probability as a function of the occupation probability.
