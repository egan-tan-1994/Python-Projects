This simulates the fractional-dosing irradiation of cancer cells via stochastic modeling using the chemical Langevin equation.
Stochasticity as considered from various sources: radiation dosing, cell proliferation, repair, mutation, and death, while considerng different sources of radiation.
This model can help in selecting the best type of radiation used for cancer treatment, as well as implementing better dosimetry protocols to effectively treat cancer
by killing cancer cells, while harming as few healthy cells as possible.
